# Frontend for Combine Project

This is a minimal static frontend (HTML/CSS/JS) that connects to the backend API exposed at `http://localhost:3000/movies`.

Quick start

1. Start the backend from the project root:

```bash
npm run hari
# or
node main.js
```

2. Open the frontend file in your browser:

- Open `frontend/index.html` directly, or
- Serve the `frontend` folder with a static server (e.g. `npx serve frontend`)

The UI supports listing movies, creating new entries, editing, deleting, and searching by name.
